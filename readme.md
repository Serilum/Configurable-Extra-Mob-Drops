<p><span style="font-size:18px">Configurable Extra Mob Drops is a server-side mod which adds the ability to add extra items to drop on death for any entity. On first load, it generates a config file. You can add items manually, or use the commands in-game. Any items with NBT tags such as enchantments and written books are supported. Modded entities are also supported, just make sure to remove or rename the config file in between adding mods which add custom mobs. If the file does not exist, it'll be generated with all possible entity types.<br><br>The config file generated is located in&nbsp;'<em>./config/configurableextramobdrops/</em>' named&nbsp;'<em>mobdropconfig.txt</em>'. The drop chances are set via the command or can be changed in the config file as well.<br></span><br><br><span style="font-size:24px"><strong>Commands:</strong></span><br><span style="font-size:14px"><em>/cemd usage</em></span> - Shows the usage of commands.<br><span style="font-size:14px"><em>/cemd list</em></span> - Lists all entity names which can be used in <em>addhand</em> and <em>cleardrops</em>.<br><span style="font-size:14px"><em>/cemd reload</em></span> - Reloads all manual changes to the mobdropconfig file.<br><span style="font-size:14px"><em>/cemd addhand &lt;entity-name&gt;</em></span> - Adds the item held in the main hand as a drop for the entity named &lt;entity-name&gt; with a 100% dropchance.<br><span style="font-size:14px"><em>/cemd addhand &lt;entity-name&gt; &lt;drop-chance&gt;</em></span> - Adds the item held in the main hand as a drop for the entity named &lt;entity-name&gt; with a dropchance in between 0 and 1.0.<br><span style="font-size:14px"><em>/cemd cleardrops &lt;entity-name&gt;</em></span> - Clears all extra drops for the entity named &lt;entity-name&gt;.<br><br><em>All commands require cheat-access.</em><br><br><br><span style="font-size:24px"><strong>Images:</strong></span><br><span style="font-size:14px">The <em>/cemd usage</em> command:</span><br><picture><img src="https://github.com/ricksouth/serilum-mc-mods/raw/master/cdn/configurable-extra-mob-drops/a.jpg" width="1000" height="538"></picture><br><br><span style="font-size:14px">The <em>/cemd list</em> command.
